# GeoCercaPM-Ionic-App

**Descripción**  
App móvil para control de ubicación en tiempo real usando geocercas. Diseñada para monitorear zonas geográficas específicas y notificar cuando un dispositivo entra o sale de la zona.

**Tecnologías**  
- Ionic 7 + Angular 17  
- TypeScript  
- REST API externa (Node.js + Express)  
- Base de datos MySQL  
- Integración de geocercas con Geolocation y Geofence Plugin
